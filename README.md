# openGDE
An open-source electrochemical test platform. Published in the Australian Journal of Chemistry. Manuscript DOI: [10.1071/CH25175](https://doi.org/10.1071/CH25175)

Repository release v1.0.0 available: [![DOI](https://zenodo.org/badge/711716960.svg)](https://doi.org/10.5281/zenodo.18322570)

## Information
- Designed in [openSCAD](https://openscad.org/), with code written as extensibly as possible for people to adapt for their needs.
- STLs sliced using [CURA](https://ultimaker.com/software/ultimaker-cura/) 4.12.1 or [OrcaSlicer](https://www.orcaslicer.com/) 2.3.1
- Printed on a modified Ender 3 (running [Octoprint](https://octoprint.org/) 1.9.3, all metal hot-end and glass bed for most prints).
- Printed with PLA, PETG, TPU, TPE.

## Additional hardware
- M3 nuts and bolts (12 mm).
- Standard nylon tube for pneumatic applications, 4mm OD with 2.5mm ID (cut to length and flattened with heat treatment and mold).
- Commercial membrane (tested with Nafion and Fumasep FAD)
